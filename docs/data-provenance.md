# Data Provenance

## Included Sample Data

- `sample-data/sir_covid/al_covid.csv`

This file is a small example dataset used by the SIR fitting example. It is suitable for reproducible demonstrations in this repository.

## Data Policy

- Keep only small, non-sensitive example data in the repository.
- Do not commit raw private data, local exports, or unpublished research datasets.
- Larger source datasets should be referenced in documentation instead of stored directly in the repository unless redistribution is clearly permitted.

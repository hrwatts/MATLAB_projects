# Bifurcation

This project explores period doubling and chaotic behavior that can appear when Euler's method is used on nonlinear autonomous differential equations.

## Files

- `bifurcation.mlx`: main live script for the project
- `plotb.m`: helper for generating a bifurcation diagram by sweeping step sizes
- `plotd.m`: helper for related diagram or trajectory plotting
- `README.source.md`: original branch README

## Usage

Open `bifurcation.mlx` in MATLAB for the guided walkthrough. The helper scripts are intended to be run from the same folder so they can access the local project files directly.

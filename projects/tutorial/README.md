# MATLAB Tutorial

This folder groups the tutorial-oriented materials from the original `tutorial` branch.

## Files

- `matlab_tutorial.mlx`: introductory live script walkthrough
- `MATLAB_principles.mlx`: live script covering MATLAB programming concepts
- `exports/MATLAB_principles.html`: HTML export of the principles material
- `exports/MATLAB_principles.pdf`: PDF export of the principles material
- `README.source.md`: original branch README

## Usage

Open the `.mlx` files in MATLAB to view and run the tutorial content interactively. The files in `exports/` provide static rendered versions of `MATLAB_principles` for readers who do not want to open the live script first.
